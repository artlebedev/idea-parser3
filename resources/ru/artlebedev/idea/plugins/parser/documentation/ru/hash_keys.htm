_keys. Список ключей хеша

^хеш._keys[]   
^хеш._keys[имя столбца]   [3.2.2]

Метод возвращает таблицу (объект класса table), содержащую единственный столбец, где перечислены все ключи хеша (начиная с версии 3.4.0 порядок ключей в полученной таблице соответствует порядку добавления элементов в хеш, до этой версии - порядок не определен).
Имя столбца - «key» или переданное имя столбца.

Пример
$man[
   $.name[Вася]
   $.age[22]
   $.sex[m]
]
$tab_keys[^man._keys[]]
^tab_keys.save[keys.txt]

Будет создан файл keys.txt с такой таблицей:
key
name
age
sex
