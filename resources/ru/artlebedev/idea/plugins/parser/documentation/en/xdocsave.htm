save. Saving document to file

^document.save[path]
^document.save[path;Document_to_text_conversion_parameters]

This method saves document to a text file. One can also specify conversion_parameters.
By default, method will create XML-representation of the document with header <?xml … ?> (one can also disable this header by specifying relevant parameter).

Path-path to file.

Example
$document[^xdoc::create{<?xml version="1.0" encoding="windows-1251" ?>
<document>
string1<br />
string2<br />
</document>}]
^document.save[saved.xml]
