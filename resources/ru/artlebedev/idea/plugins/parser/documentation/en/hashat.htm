_at. Element access by index   [3.4.1]

^hash._at(index) 
^hash._at(-index) 
^hash._at[first] 
^hash._at[last] 

The method returns value with specified index (starting from zero), so ^hash._at(0) is equal to ^hash._at[first].
For negative index it is calculated from the end of the hash, so ^hash._at(-1) is equal to ^hash._at[last].
