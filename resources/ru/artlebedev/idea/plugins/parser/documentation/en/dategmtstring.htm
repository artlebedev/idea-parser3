gmt-string. Converting date to string in RFC 822 format [3.2.3]

^date.gmt-string[]

This method convert date to string in RFC 822 format (Fri, 23 Mar 2001 09:32:23 GMT).
Usually you don't need to do anything and Parser convert date to such string automatically (for example when you set HTTP-response header: $response:expires[^date::now(+1)]). But somethime (when you generate RSS feed for example) this method can be usable.
