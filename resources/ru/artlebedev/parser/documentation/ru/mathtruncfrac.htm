trunc, frac. ќперации с целой/дробной частью числа

^math:trunc(число)
- возвращает целую часть числа
^math:frac(число)
- возвращает дробную часть числа


ѕример
^math:trunc(85.506)
- получаем 85
^math:frac(85.506)
- получаем 0.506
