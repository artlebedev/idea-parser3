_count. Количество ключей хеша

^хеш._count[]   

Возвращает количество ключей хеша.

Пример
$man[
   $.name[Вася]
   $.age[22]
   $.sex[m]
]
^man._count[]

Вернет: 3.

В выражениях числовое значение хеша равно количеству ключей:
^if($man > 2){больше}
