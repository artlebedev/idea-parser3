Numeric literals

Numeric literals can have the following possible forms:

1
-8
(integer)

1.23
-4.56
(fractional)

1E3 equals to 1000
-2E-6 equals to -0.000002
(so-called scientific notation, format: stagnatEexponent)

0xA8 equals to  168
(integer in hexadecimal code)

Note: case-insensitive.
