Json class   [3.4.1]

The class is designed for working with JSON (JavaScript Object Notation).

JSON is a lightweight text-based open standard designed for human-readable data interchange. It is based on a subset of the JavaScript Programming Language, Standard ECMA-262 3rd Edition - December 1999.
