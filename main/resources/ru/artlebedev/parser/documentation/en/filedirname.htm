dirname. Path to file

^file:dirname[filespec]

Retrieves path from path and filename to file/directory (filespec).

Example
#filename
^file:dirname[/a/some.tar.gz]
#directory's name…
^file:dirname[/a/b/]

In both cases the result will be:
/a
