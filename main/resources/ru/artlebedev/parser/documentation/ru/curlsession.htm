session. Создание сессии

^curl:session{код}

Метод создаёт cURL-сессию. Код метода обрабатывается Parser, позволяя работать с удалённым сервером в рамках одной установленной HTTP-сессии, сли удалённый сервер поддерживает keep-alive.
Внутри сессии могут быть установлены общие опции и сделано несколько вызовов метода загрузки файла.

Пример
^curl:session{
   ^curl:options[
      $.url[https://store.artlebedev.ru/]
      $.charset[UTF-8]
      $.timeout(10)
   ]

   $file1[^curl:load[
      $.url[https://store.artlebedev.ru/login/]
      $.form[
         $.Username[$form:login]
         $.Password[$form:password]
         $.btnSubmit[Войти]
      ]
   ]]

   $file2[^curl:load[]]
}
