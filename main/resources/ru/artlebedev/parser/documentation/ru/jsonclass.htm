Json (класс) [3.4.1]

Класс для работы с JSON (JavaScript Object Notation).

JSON - это альтернатива традиционным форматам (обычный текст или XML), которые используются при обмене данными между сервером и клиентом. В отличие от XML и XML-совместимых языков, которые требуют синтаксического анализа, определения JSON могут быть просто включены в сценарии JavaScript.
JSON также является текстовым форматом обмена данными и может довольно легко читаться людьми.
