exp, log, log10. Логарифмические функции

^math:exp(число)
- экспонента по основанию e
^math:log(число)
- натуральный логарифм
^math:log10(число)
- десятичный логарифм


Методы вычисляют значения логарифмических функций от заданного числа

Примечание (если вы совсем забыли родную школу): логарифм по произвольному основанию base вычисляется как log(число)/log(base).
