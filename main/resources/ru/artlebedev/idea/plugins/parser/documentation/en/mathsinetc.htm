sin, asin, cos, acos, tan, atan. Trigonometric functions

^math:sin(radians)
sine
^math:asin(number)
arc sine
^math:cos(radians)
Cosine
^math:acos(number)
arc cosine
^math:tan(radians)
Tangent
^math:atan(number)
arc tangent


These methods calculate values of trigonometric functions of a specified number.

Example
^math:cos(^math:radians(180))

will return -1.
